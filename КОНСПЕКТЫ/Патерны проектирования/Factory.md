# Проблема котороую решает Factory:
> Создание объекта становится сложнее, чем его использование

Причины:
- Разные типы объектов
- Сложная логика выбора
- Зависимость от входных данных
- Необходимость скрыть new

# Суть
Factory инкапсулирует логику создания объекта
это дает
- изоляцию измеенний
- читаемость
- тестируемость
# Когда Factory оправдан
- есть несколько вариантов создания объекта
- тип объекта определяется во время выполнения
- создание от бизнес правил
