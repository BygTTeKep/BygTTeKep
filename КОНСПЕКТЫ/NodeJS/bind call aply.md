все 3 метода управляют контекстом выполенния функции

bind - возращает новую функцию с привязанным this
```js
const newfunc = func.bind(newThis)
```

call - сразу исполняет функцию с указаным this, где аргументы передаются через запятую
apply - тоже самое, но аргументы тут передаются в виде списка
```js
func.call(newThis, 1,2,3,4)
func.apply(newThis, [1,2,3,4])
```
