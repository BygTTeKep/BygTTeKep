Атомики - это примитив из пакет sync/atomic, которые позволяют безопасно работать с переменными из нескольких горутин без Mutex.

Ключевое свойство - операции выполняются неделимо. Это означает что ни одна другая горутина не увидет промежуточное состояние операции.

Проблема которую решают атомики - race condititon.

Атомики работают только с простыми операциями.

Основные операции
```go
var p int64 = 0
atomic.AddInt64(&p, 1) //атомарное увеличение 
atomic.AddInt64(&p, -1) //атомарное уменьшение
atomic.LoadInt64(&p) // атомарное чтение
atomic.StoreInt64() //атомарная запись
CAS

```
