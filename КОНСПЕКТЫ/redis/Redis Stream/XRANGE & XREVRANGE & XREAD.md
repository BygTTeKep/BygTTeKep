**XRANGE** — чтение по возрастанию ID:
```
XRANGE orders - +
```
`-` и `+` обозначают диапазон: от самого старого до самого нового сообщения.

**XREVRANGE** — чтение по убыванию ID:

`XREVRANGE orders + -`
### Чтение новых сообщений: `XREAD`
`XREAD COUNT 2 STREAMS orders 0

- `COUNT 2` — читаем максимум 2 сообщения.    
- `0` — начиная с самого первого ID.
- Можно использовать `$` для получения **только новых сообщений**, которые появятся после вызова команды.
