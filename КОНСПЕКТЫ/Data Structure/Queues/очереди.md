Очереди это структура данных работабщая по принципу FIFO
# базовые операции
enqueue- добавление элемента в конец
dequeue - извлечь элемент с начала
peek - посмотреть первый элемент
# Основные реализации
## Очередь на массиве
Идея
используется массив фиксированного размера или расширяемый
два индекса head и tail
индексы холят по кругу

## Очередь на односвязном списке
head указывает на первый элемент
tail на последний элемент
вставка в tail извлечение head

# Виды по реализации
## simple fifo
## bounded queue
очередь с ограниченым размером, при переполнении
- блокируем
- дропаем
- применяем blackpressure
## priority queue
элементы выходят не по времени, а по приоритету
обычно реализуется через heap а не fifo
## delay / sheduled queue
элемент можно извлечь только после заданного времени

## Dequeue (Double-ended queue)
можно добавлять и удалять с двух концов 
используется в LRU cache
