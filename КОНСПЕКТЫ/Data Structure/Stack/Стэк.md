Stack - это структура данных работающая по принципу LIFO


```go
package main

type Struct[T any] struct {
	data []T
}

func (s *Struct[T]) push(value T) {
	s.data = append(s.data, value)
}

func (s *Struct[T]) pop() (T, bool) {
	var zero T
	if len(s.data) == 0 {
		return zero, false
	}
	v := s.data[len(s.data)-1]
	s.data = s.data[:len(s.data)-1]
	return v, true
}
```

**Свойства:**

- `push/pop` — O(1) amortized
- excellent cache locality
- минимум аллокаций
**Используется почти всегда**, если нет требований к lock-free.
